<h2>WeatherApp</h2>

<p>Simple weather app with animated background. You can check weather using your's current localization or by typing city name.</p>

<h4>Used technologies</h4>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>Bootstrap 4.0</li>
  <li>Vanilla JS</li>
  <li>jQuery</li>
</ul>

<h4>Used APIs</h4>
<ul>
  <li>GoogleMap</li>
  <li>OpenWeatherMap </li>
</ul>

<p>Getting city name taken from <a href="https://stackoverflow.com/questions/6548504/how-can-i-get-city-name-from-a-latitude-and-longitude-point">Stackoverflow</a> using solution created by user <a href="https://stackoverflow.com/users/3894567/sanchit-gupta">Sanchit Gupta</a></p>
